# my_game
Repository created to develop a basic 2D RPG game.
The idea is to create a simple and scalable game in 2D, probably pixelart-like.

# Notes

Every game starts the same:

    while game_on:
        read_input()
        update_state()
        draw_state()

The game can have several states:
    overworld
    battle
    menu
    dialogue


# Structure

**Core**

    Game engine

**Entities**

    Entities like player, npcs and mobs.

**Systems**

    Combat, dialogs, etc.

**World**

    World

**Assets**

    Images

**Utils**

    Utilities
